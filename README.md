Project Summary: Online Banking Transaction System
This project is designed to track and manage online banking transactions efficiently and securely. '
It leverages the principles of Object-Oriented Programming (OOP) in Python to ensure modularity, scalability, and maintainability.

Key Highlights:
- Transaction Tracking: Records deposits, withdrawals, transfers, and account balances.
- OOP Design: Implements classes and objects to model core banking entities such as Account, Transaction, and Customer.
- Encapsulation & Abstraction: Protects sensitive data while exposing only necessary functionalities.
- Inheritance & Polymorphism: Enables code reuse and flexibility for different transaction types.
- Scalability: Provides a foundation for extending features like authentication, reporting, and integration with external banking APIs.
